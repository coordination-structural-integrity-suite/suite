# Examples

Paste-and-go prompts for using the Precision-First Design Standard with any AI assistant.

| Prompt | Use it on |
|---|---|
| [General Audit](audit-general.md) | Any document, specification, or standard |
| [Governance Audit](audit-governance.md) | DAO proposals, bylaws, charters, policies, constitutions |
| [Software Audit](audit-software.md) | API contracts, requirements docs, schemas, technical designs |

Each file contains a self-contained prompt. Copy everything below the line in the file, paste it into any AI chat (Claude, ChatGPT, Gemini, or similar), and attach the document you want audited. The AI will produce a precision deficit map: a structured list of places where the document's terms, rules, or processes are not precise enough to be enforced or tested by an independent observer.

No prior knowledge of the standard is required. The prompts carry everything the AI needs.

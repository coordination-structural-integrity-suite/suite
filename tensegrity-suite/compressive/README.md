# Tensegrity Compressive Standards

Seven standards that specify structural floors: the minimum conditions a coordination system must hold to prevent predictable failure modes. Each standard is an instrument with verifiable conditions. They specify what must not be violated; they do not specify how to organize.

**Precision-First Design Standard**: requires that every element of a coordination system increase the precision with which the system's dynamics can be observed, classified, and acted upon.

**Adverse Signal Engagement Principle Core Standard**: requires engagement with signals that contradict current models rather than suppression or reframing.

**Structural Consent Legibility Standard**: requires that consent to participation be structurally distinguishable from consent to specific terms, outcomes, and power arrangements.

**Information Asymmetry Classification Standard**: requires classification and disclosure of the structural types of information asymmetry present in a coordination system.

**Structural Power Obligation Standard**: requires that structural power arrangements across three independent dimensions (authority, coordination, specialization) be legible and contestable by participants who hold less of them.

**Regenerative Obligation Standard**: requires that extraction of contributor capacity be matched by regenerative return satisfying three conditions simultaneously: non-fungibility, proximity, and embeddedness.

**Coordination Scaling Standard**: requires that coordination systems install the structural conditions each Dunbar-scale threshold requires before crossing it.

# Coordination Structural Integrity Suite Assessment — Stage 5: External Frame Assessment
**Coordination Structural Integrity Suite Assessment, Stage 5 of 6**

Use this prompt after completing Stage 4. External frame assessment evaluates the arrangement from outside its own logic — asking questions the suite standards cannot ask because those questions require a vantage point the standards do not occupy. The four assessments in this stage complement the standards findings; they do not replace them.

You are running Stage 5 of the Coordination Structural Integrity Suite Assessment. This stage runs four external frame assessments: ecological fit, counterfactual, legitimacy class, and value instantiation. Each assesses the arrangement from a vantage point the suite standards do not occupy. Use the Stage 0 context summary and Stage 4 gap map provided as inputs throughout.

**A note on insider limitation.** Three of these four assessments — ecological fit, counterfactual, and legitimacy class — are significantly harder to complete honestly from inside the arrangement. The parties most familiar with an arrangement are also most adapted to its framing. Where an external reviewer is available, involve them. Where one is not, name the limitation explicitly in the output: state who completed this assessment and that insider adaptation may affect the findings.

Run Assessments 1, 2, and 3 in any order. Run Assessment 4 last; it draws on the prior three.

## Assessment 1: Ecological Fit

Ecological fit asks how the arrangement interacts with the broader ecosystem it sits in — not just with its defined party list, but with adjacent institutions, alternative approaches, and the conditions the broader ecosystem depends on. An arrangement can function correctly in isolation while damaging the conditions that make it possible. The suite standards assess quality of execution within the arrangement; they do not assess what the arrangement does to the ecosystem outside it.

Ask and answer each of the following:

**Crowding.** Does this arrangement crowd out alternatives in its domain? Name the alternatives — coordination approaches, institutional forms, or practitioners — that do not exist or cannot sustain themselves because this arrangement holds the domain. The Stage 0 failure mode profile is relevant here: FM4 (quota-based supply restriction), FM6 (innovation suppression), FM8 (war on illegitimate producers), and FM11 (epistemic monoculture) are each a form of ecological crowding. If any of those are active, state specifically what is being crowded out.

**Dependencies.** Does this arrangement depend on conditions that adjacent institutions must provide? Name those conditions and the institutions that provide them. If any of those institutions changed or failed, what would break in this arrangement? Dependencies are distinct from the external legitimation precondition from Stage 0: legitimation is a condition for the arrangement's durability; dependencies are conditions for its function.

**Externalities.** Does this arrangement produce costs or risks that the broader ecosystem must absorb without receiving compensation? This extends the affected party scope from Stage 0 to the ecosystem level: not just specific parties harmed by specific decisions, but distributed effects on the ecosystem's capacity to function. Ecological externalities are often invisible to the arrangement because no single party bears enough of the cost to contest it.

**Ecosystem value.** What does the broader ecosystem gain from this arrangement's continued existence? State this specifically: not "coordination," but what specific coordination gap this arrangement fills that would otherwise remain unfilled or be filled worse. This is the affirmative ecological case; the counterfactual assessment will test it.

## Assessment 2: Counterfactual

Counterfactual assessment asks whether this arrangement is better than its absence, and better than feasible alternatives. This is a question economists and policy analysts ask routinely that coordination designers rarely ask. The suite standards assess whether the arrangement functions; they do not ask whether functioning is better than not having it.

Ask and answer each of the following:

**Coordination problem identification.** What specific coordination problem is this arrangement solving? State it precisely — not the nominal purpose, but the structural problem: what would fail to coordinate, or coordinate badly, without this arrangement? Be specific about which parties would bear that cost.

**Absence assessment.** If this arrangement did not exist, would that coordination problem remain unsolved, or would something else fill the gap? Name what would fill it and how. If the answer is "nothing would fill it," state what the cost of that absence would be, specifically, for which parties.

**Net assessment.** Weighing the ecosystem value from Assessment 1 against the costs the arrangement creates — the affected party scope from Stage 0, the crowding and externalities findings above — is this arrangement net-positive for its domain? State the basis for the assessment, not just the conclusion.

**Feasible alternatives.** Name at least one alternative design that would address the same coordination problem with a different failure mode profile. A feasible alternative is one that: (a) addresses the same coordination problem, (b) could realistically be established given the ecosystem's current conditions, and (c) produces a different distribution of compressive and generative preconditions. If no feasible alternative exists, state why — what structural conditions make this form the only viable option.

**Foreclosure.** Does this arrangement's existence prevent better alternatives from emerging? External legitimation combined with insider authority makes some arrangements structurally self-perpetuating even when better alternatives are available. The tensegrity configuration from Stage 0 is the primary input here: a T6 configuration with low-revocability legitimation is a strong indicator of foreclosure. State the foreclosure finding with the tensegrity configuration as evidence.

## Assessment 3: Legitimacy Class

Legitimacy class asks on what basis this arrangement claims authority over the parties it coordinates with or affects, and whether that basis is sound. Different legitimacy claims have structurally distinct failure modes. An arrangement claiming democratic legitimacy fails differently from one claiming expert legitimacy. The suite does not currently ask which class applies; this assessment does.

**The five legitimacy classes and their primary failure modes:**

*Expert legitimacy* — authority claimed through superior knowledge or competence. Primary failure modes: credential inflation (the credential substitutes for competence rather than evidencing it), guild capture (the competence-defining group writes criteria that favor incumbents over genuine competence), epistemic monoculture (the definition of expertise narrows over time to serve a shrinking set of practitioners), and knowledge monopolization (the expert group controls not only who counts as expert but what counts as knowledge in the domain). The professional certification body and craft guild forms operate primarily through expert legitimacy claims.

*Democratic legitimacy* — authority claimed through representative mandate, community consent, or voting. Primary failure modes: coordination theater (the appearance of democratic process without effective community voice), manufactured consent (the agenda-setting power of dominant actors shapes what choices are available before voting occurs), participation asymmetry (the formal mechanism is democratic but effective decision-making is concentrated in a subset of participants), and preference aggregation without deliberation (outcomes reflect aggregated preferences rather than considered judgment under conditions of good information). Token-weighted governance bodies nominally claim democratic legitimacy but operate through capital-weighted topologies that produce participation asymmetry at scale.

*Traditional legitimacy* — authority claimed through historical precedent, established custom, or recognized practice. Primary failure modes: precedent bias against innovation (what was done becomes what must be done), conservatism as authority (persistence is treated as evidence of validity), and historical injustice perpetuation (precedents established under unjust conditions are maintained as authoritative).

*Market legitimacy* — authority claimed through voluntary participation and revealed preference. Primary failure modes: information asymmetry gaming (apparent consent operates under conditions where parties cannot assess what they are consenting to), first-mover advantage as false legitimacy (early market position is treated as a legitimation signal rather than as path dependence), and constrained consent (revealed preference under conditions of limited alternatives does not constitute genuine voluntary endorsement).

*Procedural legitimacy* — authority claimed through fair and transparent process. Primary failure modes: procedural capture (the process is formally correct but controlled by incumbents — the OOXML ISO/IEC JTC1 case is the documented instance), rule lawyering (procedural formalism substitutes for substantive fairness), and process-outcome disconnection (procedurally correct processes produce outcomes that are not substantively legitimate because the process design advantages certain parties).

Ask and answer each of the following:

**Claimed legitimacy class.** What legitimacy class does this arrangement claim, explicitly or implicitly? Most arrangements claim more than one; name all that apply. Use both the arrangement's own framing and the Stage 0 institutional form classification together — the institutional form often makes the primary legitimacy claim visible even when the arrangement does not state it explicitly.

**Soundness assessment.** Is each legitimacy claim sound given the arrangement's actual structure? For each claimed class, does the arrangement's structure satisfy the conditions for that legitimacy class, or does it claim the legitimacy without meeting the structural conditions? State the finding per class.

**Active failure modes from the legitimacy class.** For each claimed legitimacy class, identify which of that class's primary failure modes are active in this arrangement. Cross-reference with the Stage 0 failure mode profile: the active compressive failure modes (FM1-FM11) are often legible as expressions of legitimacy class failure as well as structural failure — naming both framings gives the strongest diagnostic picture.

## Assessment 4: Value Instantiation

Value instantiation asks what values this arrangement actually produces through its incentive structures, regardless of what it claims. This is distinct from correctness of design (which asks whether the design achieves its stated purpose) and from quality of execution (which asks whether it executes its design well). Value instantiation asks what is produced as a structural effect of the arrangement's existence, including effects that were not designed and are not stated.

Run this assessment after completing Assessments 1, 2, and 3. It draws on all three.

**Stated values.** Name the values this arrangement claims to instantiate. Use the arrangement's founding documents, nominal purpose, and stated goals. Do not interpret; name what was claimed.

**Incentive structure audit.** For each structural element of the arrangement — the revenue routing, the admission or participation mechanism, the authority topology, the exit architecture — state what behavior it rewards and what behavior it penalises. The Stage 0 compressive precondition profile is the primary input: each precondition names a structural element whose incentive effects can be audited. The Stage 0 generative precondition profile names the absences: what the arrangement does not reward and cannot produce because the structural conditions for it are absent.

**Unstated instantiations.** What values are produced as structural effects that are not stated design values? The Stage 0 failure mode profile is the clearest signal: each active failure mode names a value being instantiated regardless of what was stated. FM1 (rent extraction) instantiates incumbent protection. FM3 (dynastic capture) instantiates social reproduction. FM6 (innovation suppression) instantiates stability over adaptability. FM9 (persistence despite inefficiency) instantiates institutional continuity over performance accountability. FM11 (epistemic monoculture) instantiates definitional uniformity over knowledge diversity. FM12 (constitutional capture) instantiates incumbent authority as constitutional authority. FM13 (externality blindness) instantiates insider interests as the measure of legitimate coordination. FM15 (nominal purpose shield) instantiates stated purpose as sufficient evidence of actual purpose. State the value each active failure mode is producing; do not just list the failure mode names.

**Divergence map.** Where do stated values and instantiated values diverge? Name each divergence specifically: "[Stated value X] is contradicted by [Instantiated value Y] because [the structural mechanism that produces Y while claiming X]." The divergence map is the core finding of the value instantiation assessment. A stated value with no divergence entry is a finding too — it means the incentive structure and the stated value are aligned, which is worth naming explicitly.

**Revised counterfactual.** Does the value divergence change the counterfactual net assessment from Assessment 2? If the arrangement instantiates values at odds with its stated purpose, does that affect whether it is better than its absence or better than feasible alternatives? If the net assessment changes, state the revised finding and the basis for the revision.

## Synthesis

After completing all four assessments, produce the external frame synthesis. This synthesises the findings into a unified external assessment that supplements — it does not replace — the standards gap map from Stage 4.

**What does the external frame assessment reveal that the standards gap map did not show?** Name specifically: not "things the standards couldn't catch" but the specific findings produced by the four assessments that have no equivalent in the Stage 4 gap map.

**Where do the external frame findings and the standards findings converge?** Convergence is a stronger indicator than divergence: if the same gap appears in both the standards findings and the external frame assessment, it is more structurally significant than a gap that appears in only one. Name every convergence point.

**What is the single most important finding from the external frame assessment for this arrangement's current situation?** One sentence. This becomes the opening line of the Stage 6 advancement session.
